# TechCore-DIW-CO

Proyecto estático (HTML/CSS) para la asignatura de Diseño de Aplicaciones. Contiene una maqueta de tienda llamada "TechCore" con páginas HTML, estilos en CSS y recursos gráficos.

**Características principales**

- Páginas estáticas: `index.html`, `catalogo.html`, `carrito.html`, `contacto.html`, `mi-cuenta.html`, `politica-privacidad.html`, `quienes-somos.html`.
- Estilos centralizados en `css/styles.css`.
- Recursos gráficos en la carpeta `imagenes/`.

**Estructura del proyecto**

- `html/` : Todas las páginas HTML del sitio.
- `css/`  : Hoja(s) de estilo (actualmente `styles.css`).
- `imagenes/` : Imágenes y logos usados en la maqueta.

**Cómo ver el proyecto localmente**

Opción 1 — Abrir directamente en el navegador:
- Abra el archivo `html/index.html` con su navegador (doble clic o "Abrir con...").

Opción 2 — Servir con un servidor local (recomendado para rutas relativas y recursos):

En PowerShell (desde la raíz del proyecto):

```powershell
python -m http.server 8000
```

Luego abrir en el navegador:

```
http://localhost:8000/html/index.html
```
